Visual Studio Code

Instalación

  1. Ve a la página de Microsoft Visual Studio Code en Academic Software y le damos clic en 'Descargar Visual Studio Code'. Esto dependiendo del sistema operativo que estemos utilizando en este caso sería un Windows 10 Pro, le tendríamos que dar a Windows 10, 11.
  2. Abre el archivo de instalación .exe en tu carpeta de descargas para iniciar la instalación.
  3. Lee y acepta el acuerdo de licencia. Haz clic en Next para continuar.
  4. Puedes cambiar la ubicación de la carpeta de instalación o mantener la configuración predeterminada. Haz clic en Next para continuar.
  5. Elige si deseas cambiar el nombre de la carpeta de accesos directos en el menú Inicio o si no deseas instalar accesos directos en absoluto. Haz clic en Next.
  6. Selecciona las tareas adicionales, por ej. crear un icono en el escritorio o añadir opciones al menú contextual de Windows Explorer. Haz clic en Next.
  7. Haz clic en Install para iniciar la instalación.
  8. El programa está instalado y listo para usar. Haz clic en Finish para finalizar la instalación y lanzar el programa.

Configuración Inicial

  • Choose your theme: Permite personalizar los colores del editor seleccionando un tema visual que sea cómodo para tus ojos.
  • Rich support for all your languages: Configura el soporte para diferentes lenguajes de programación. VS Code es compatible con extensiones que añaden características específicas como resaltado de sintaxis, autocompletado, etc.
  • Tune your settings: Permite personalizar cómo funciona VS Code. Puedes cambiar fuente, tamaño del texto, formato del código, etc., desde el archivo settings.json.
  • Sync settings across devices: Habilita la sincronización en la nube para usar las mismas configuraciones, extensiones y temas en diferentes computadoras.
  • Unlock productivity with the Command Palette: Introduce la Command Palette, una herramienta poderosa para ejecutar cualquier comando en VS Code sin usar el ratón.
  • Open up your code: Abre una carpeta o archivo donde tengas tu código fuente.
  • Watch video tutorials: Acceso a videos oficiales que explican cómo usar VS Code de forma efectiva.

Guía para las opciones superiores y laterales

Menú Superior (Barra de Menús):

  • File (Archivo): Crear, abrir, guardar archivos o carpetas y salir del editor.
  • Edit (Editar): Acciones comunes como cortar, copiar, pegar, deshacer y buscar/reemplazar texto.
  • View (Vista): Mostrar/ocultar paneles como el explorador, terminal, extensiones o depurador.
  • Go (Ir): Navegar entre archivos, líneas de código, funciones o símbolos en tu proyecto.
  • Run (Ejecutar): Iniciar y gestionar la ejecución o depuración de tu código.
  • Terminal: Abre una terminal integrada para ejecutar comandos sin salir del editor.
  • Help (Ayuda): Acceso a documentación, tutoriales y detalles de la versión de VS Code.

Barra Lateral Izquierda (Herramientas principales):

  • Explorer (Carpeta): Administra y abre los archivos y carpetas de tu proyecto.
  • Search (Lupa): Busca texto en todos los archivos del proyecto y realiza reemplazos.
  • Source Control (Git): Gestiona repositorios Git (commits, ramas, etc.).
  • Run and Debug (Reproducir + Bug): Depura y ejecuta aplicaciones configuradas en tu proyecto.
  • Extensions (Bloque): Instala y gestiona extensiones para lenguajes o herramientas adicionales.

NetBeans

Instalación

  1. Acceder al enlace de descarga: NetBeans Apache
  2. Descargar la última versión acorde al Sistema Operativo, actualmente es la 23.
  3. Seleccionar el proveedor de descarga.
  4. Ejecutar el instalador: La instalación por defecto incluye las características básicas de Java SE, Java Enterprise Edition, Html/Javascript y Php. Puedes personalizar las opciones.
  5. Aceptar términos de licencia.
  6. Seleccionar ubicación y versión del JDK.
  7. Haz clic en Install para iniciar la instalación.

Configuración Inicial

  • New Project: Crear un proyecto nuevo (Java, PHP, etc.).
  • Open Project: Abrir un proyecto existente.
  • New File: Crear un archivo nuevo (clases, interfaces, etc.).
  • Open File: Abrir un archivo existente en el proyecto.
  • Go to File: Buscar un archivo dentro del proyecto.
  • Show Dashboard: Mostrar un panel con información de proyectos.

Incidencia 1: El JDK es un conjunto de herramientas de desarrollo de software necesarias para construir aplicaciones Java. NetBeans IDE depende del JDK para funcionar correctamente. El mensaje de error indica que el JDK no está instalado en la computadora o que el instalador no puede encontrarlo.

Solución: Para solucionar este problema, debes descargar e instalar el JDK 8 o una versión más reciente desde el sitio web de Oracle. Una vez que el JDK esté instalado, puedes volver a ejecutar el instalador de NetBeans IDE JDK Development.

IntelliJ IDEA

Instalación

  1. Vaya a este enlace para obtener el software IntelliJ, haga clic en la opción Descargar Software IntelliJ.
  2. Ejecutar el instalador.
  3. Puedes cambiar la ubicación de la carpeta de instalación o mantener la configuración predeterminada. Haz clic en Next para continuar.
    • Create Desktop Shortcut: Crea un acceso directo en el escritorio para abrir IntelliJ IDEA fácilmente.
    • Update PATH Variable: Permite usar IntelliJ desde la terminal con comandos.
    • Update Context Menu: Agrega la opción "Abrir carpeta como proyecto" al menú contextual al hacer clic derecho.
    • Create Associations: Asocia tipos de archivos como .java, .kt, .gradle, etc., para que se abran automáticamente con IntelliJ IDEA.
  4. El instalador solicita al usuario que elija una carpeta en el menú de inicio de Windows donde se almacenarán los accesos directos al programa.

Configuración Inicial

  • Projects: Crear o abrir un proyecto existente.
  • Remote Development: Conectarse a un entorno remoto (SSH, WSL, Dev Containers).
  • Customize: Personaliza las configuraciones de IntelliJ IDEA.
  • Plugins: Gestionar los plugins instalados.
  • Learn: Acceder a la documentación y recursos de aprendizaje.

Eclipse

Instalación

  1. Accede a Eclipse y haz clic en "Descargar 64 bits".
  2. Haz clic en el archivo “eclipse-inst-win64.exe” para iniciar la instalación.
  3. Selecciona "Eclipse IDE para Java Desarrolladores", puedes cambiar las rutas de instalación o dejarlas predeterminadas.
  4. Acepta los términos de licencia y haz clic en Install.

Configuración Inicial

Ya podrás iniciar el Launcher y comenzar a usar Eclipse para tus proyectos Java.

Compatibilidad

Asegúrate de que el proyecto sea compatible

  1. Antes de exportar, verifica que tu proyecto en NetBeans esté configurado de manera estándar (como un proyecto Java con estructura común). Eclipse y NetBeans usan formatos diferentes para almacenar configuraciones del proyecto, por lo que algunas cosas (como bibliotecas externas o configuraciones específicas) podrían necesitar ajustes manuales.

  2. Primero seleccionamos el archivo o js que queramos exportar.
  3. Luego le damos a file / export project / to zip.
  4. En build ZIP ponemos la ruta donde estaría el proyecto en zip.
  5. Luego nos pasamos a eclipse le damos a file y luego a import / general / Projects from Folder or Archive
  6. Y descomprimimos lo que sería el zip para poder importarlo a Eclipse, seleccionamos la carpeta y finalmente le damos a finish y nos habrá exportado correctamente el proyecto.

VENTAJAS Y DESVENTAJAS DE LAS DIFERENTES IDES

Características Eclipse NetBeans
Ventajas - Multiplataforma y de código abierto: Compatible con Windows, macOS y Linux.
- Altamente extensible con un sistema de plugins robusto.
- Ideal para Java, diseñado inicialmente para este lenguaje.
- Soporte para múltiples lenguajes como C/C++, Python, PHP y más.
- Gran comunidad y documentación: Amplio soporte con guías y foros. 		- Gratis y de código abierto: Compatible con Windows, macOS y Linux.
- Soporte nativo para múltiples lenguajes como Java, PHP, HTML5, JavaScript, y más.
- Interfaz sencilla y funcional: Ideal para principiantes.
- Integración con herramientas de desarrollo como Maven, Gradle y Git.
- Depurador visual muy intuitivo y fácil de usar.
Inconvenientes - Interfaz anticuada en comparación con otros IDE.
- Pesado y lento: Puede consumir muchos recursos en proyectos grandes.
- Curva de aprendizaje alta para personalización y configuración inicial. 		- Menos extensible en comparación con Eclipse.
- Puede volverse lento en proyectos complejos.
- Menor comunidad de soporte que Eclipse o IntelliJ IDEA.

Características IntelliJ IDEA Visual Studio Code
Ventajas - Interfaz moderna y atractiva, fácil de usar incluso para principiantes.
- Rendimiento superior, rápido y eficiente para proyectos grandes.
- Características inteligentes como autocompletado avanzado y refactorización precisa.
- Soporte integrado para frameworks como Spring, Hibernate, Kotlin, Android, y más.
- Edición de código productiva con herramientas diseñadas para optimizar el tiempo de desarrollo. 		- Gratis y de código abierto, mantenido por Microsoft.
- Ligero y rápido: Más liviano que los IDE completos.
- Extensibilidad con una amplia variedad de extensiones para lenguajes, herramientas y frameworks.
- Soporte para múltiples lenguajes como JavaScript, Python, Java, C#, Go, entre otros.
- Excelente para desarrollo web con características avanzadas para JavaScript, TypeScript, HTML y CSS.
Inconvenientes - Coste: La versión Ultimate (con todas las características) es de pago; solo la versión Community es gratuita.
- Exigente en recursos, especialmente en máquinas más antiguas.
- Menor soporte para lenguajes no relacionados con Java. 		- No es un IDE completo: Es un editor de texto avanzado, pero carece de algunas características nativas de los IDE.
- Requiere configuración para igualar las capacidades de un entorno de desarrollo completo.
